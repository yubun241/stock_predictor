# StockSight — 株価予測アプリ

**藤井工藝 / Fujii Kogei**

iPhone(Pythonista 3)上で動作する個別株の分析・予測アプリです。
外部課金なし・APIキー不要・Yahoo Financeの無料公開データのみを使用します。

---

## 特徴

- ローソク足チャート(日足3ヶ月 / 5分足2時間)の可視化
- ボリンジャーバンド(±1σ / ±2σ / ±3σ)
- RSI / MACDインジケーター
- 3営業日先までの予測ファンチャート(68% / 90% 信頼区間)
- 6モデル・アンサンブル予測エンジン「Mirage Ensemble+」
- 約60種のローソク足パターン自動検出(酒田五法含む)
- チャート上の強シグナル▲▼マーカー
- 🔍ボタンでチャートをネイティブビューア全画面表示(ピンチ拡大対応)

## 予測モデル構成

| # | モデル | 概要 |
|---|--------|------|
| M1 | ブロック・ブートストラップMC | 過去2年の実収益率を再標本化。ファットテールを保持しEWMAボラで直近環境にスケール |
| M2 | パターン類似検索 kNN | 直近10本の値動きを正規化し、過去2年の全局面と照合。最も似た25局面の「その後」の実績分布から確率を推定 |
| M3 | 自己相関 AR(1) | 収益率の系列相関からモメンタム/平均回帰のどちらの環境かを判定 |
| M4 | テクニカルスコア | RSI / MACD / SMA / BBσ乖離をスコアリングし、ロジスティック関数で確率に変換 |
| M5 | ニュースキーワード | Yahoo Financeニュース見出しを日英キーワードでポジティブ/ネガティブ解析 |
| M6 | マクロ相関 | 日経平均β・ドル円β + 夜間FX変動による翌日ドリフト推定 |

> **重みの配分:** M2(27%) M1(22%) M4(18%) M5(13%) M6(11%) M3(9%)

### 較正について

バックテスト(10銘柄×30営業日 / n=300)の実測値に基づき較正済み:

- 方向的中率: 約49〜51%(偶然水準。短期方向予測の限界)
- **68%帯内包含率: 71% / 90%帯内包含率: 91%(理想値68%/90%に近い)**

このアプリの主な価値は「上がるか当てる」ことではなく、
**「3日後にどこまで動き得るか」を較正済みの幅で示すリスク計測器**として機能することにあります。

## 使い方

### 基本操作

```
[銘柄コード入力欄]  [日足 / 5分足]  [分析ボタン]
```

1. 銘柄コード(数字4桁)を入力
   - 例: `7203` = トヨタ / `6758` = ソニー / `9984` = ソフトバンクG
   - 米国株の場合はそのまま入力: `AAPL` / `TSLA` / `NVDA`
2. 「日足」または「5分足」を選択
3. 「分析」ボタンをタップ
4. 取得・解析完了まで数秒〜15秒程度待つ

### チャートの見方

| 要素 | 説明 |
|------|------|
| 赤ローソク | 陽線(終値 ≥ 始値) |
| 青ローソク | 陰線(終値 < 始値) |
| 黄線 | SMA5 |
| 紫線 | SMA25 |
| 緑/橙/赤のバンド | ボリンジャーバンド ±1σ / ±2σ / ±3σ |
| シアンのファン | 予測範囲(中央値・68%帯・90%帯) |
| ▲緑 | 上昇シグナル(強シグナル) |
| ▼赤 | 下落シグナル(強シグナル) |
| 緑/赤/灰の数字 | ローソク足パターン番号(レポートに対応表あり) |
| 点線の縦線 | 実績と予測の境界 |

### ローソク足パターン番号

チャート上の番号をレポートと照合してください。

- **緑数字(強気)**: ハンマー / 明けの明星 / 陽の包み足 / 赤三兵 など
- **赤数字(弱気)**: 流れ星 / 宵の明星 / 陰の包み足 / 三羽烏 など
- **灰数字(中立)**: 十字線 / コマ / 窓埋め など

### チャート拡大

「🔍 拡大」ボタンをタップするとiOSネイティブビューアが開きます。

- ピンチイン・ピンチアウトで自由に拡大縮小
- ダブルタップでズーム
- 左上「完了」で元の画面に戻る
- 共有ボタンからチャート画像を保存・共有も可能

## レポートの読み方

### Mirage Ensemble+ 総合予測

```
総合上昇確率: XX%  [判定: ↗ / 信頼度: 高]
予測中央値: X,XXX
期待レンジ(68%): X,XXX 〜 X,XXX
期待レンジ(90%): X,XXX 〜 X,XXX
```

- **信頼度「高」**: 6モデルの見解が一致(一致度85%以上)
- **信頼度「中」**: ある程度割れている(参考程度に)
- **信頼度「低」**: モデルが大きく割れている(相場の不確実性が高い状態)

### マクロ相関分析 [M6]

```
日経平均: β=0.XX / 相関=+0.XX
ドル円  : β=0.XX / 相関=+0.XX → 円安メリット(輸出型)
夜間FX変動 +0.XX% → 推定ドリフト +0.XX%
```

- **円安メリット(輸出型)**: 自動車・電機など。円安時に追い風
- **円高メリット(内需/輸入型)**: 小売・食品など。円高時に追い風
- **為替中立**: FXとの連動が薄い銘柄

## 注意事項

- 本アプリはすべて**統計的推定**であり、株価の的中を保証するものではありません
- バックテスト(n=900)の結果、翌日の方向的中率は約50%(偶然水準)です
- **較正済みのリスク計測器として使用することを推奨します**
  - 「3日後に何%動き得るか」の根拠として損切り幅・ポジションサイズの設計にお使いください
